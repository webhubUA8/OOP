Task 6

Создайте класс Box, который описывает коробоку. Класс имеет свойства x, y, z, color. Где x,y,z - размеры, а color - цвет. Добавьте данному классу методы inBox который возвращает true или false, если по объему предмет влезает в коробку. Метод принимает 3 параметра x,y,z предмета, который хотят поместить в коробоку.

Task 7

Создайте экземпляр коробки box1. Заполните нужные данные. Продемонстрируйте работу метода inBox.

Task 8

Создайте наследника класса Box - SuperBox, который содержит метод inBox - данный метод работает так - вызывает метод inBox родителя и если он равен true - то возвращает true. Если false - возвращает true если два любых измерения предмета меньше чем стороны коробки ( т.е. предмет можно поместить в коробоку, но он будет выглядывать), и false если минимум 2 любых измерения предмета больше чем размеры коробки.

Task 9

Создайте на основе SuperBox экземпляр superBoxInstance и Продемонстрируйте работу метода inBox (все варианты).

Task 10

Создайте класс DrawField. Класс должен содержать три свойства h, w, id которые символизируют высоту и ширину будущего канваса и его id. Добавьте метод create(elem) - который отрисовывает canvas в указанный элемент и возвращает сам canvas.

Task 11

Наследуйтесь от класса DrawField, назовите новый класс Rect. Мутируйте метод create таким образом, чтобы он сначала рисовал канвас, после чего рисовал fillRect с параметрами указанными при запуске метода. Параметры следующие - elem, x, y, w, h - где elem - элемент куда выводится canvas, x , y координаты левой верхней точки прямоугольника, w, h - ширина и высота прямоугольника.
