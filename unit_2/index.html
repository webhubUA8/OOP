Task 1. Итак, первая задача. Мы строим интернет-магазин строительных материалов. У нас есть товары. Именно на основе товаров мы и будем строить объекты для работы и расширять их с помощью наследования. Товар имеет характеристики (свойства). Сразу уточню. Каждый объект, вы создаете в отдельном файле. Имя файла - имя объекта. Объект goods (goods.js). Goods имеет свойства: 

               price (double)
               title (string)
               image (text)
               producer (text)
               country (int)
               description (text)
               weight (kg)
               height (mm)
               width (mm)
               depth (mm)
               unit (единица продажи) (double)
и метод getCost (count) который возвращает стоимость количества единиц товара. Ввели 5 - возвратит стоимость 5 единиц. Создайте данный объект (протестируйте самостоятельно - создайте, заполните, проверьте метод).

Task 2.На основании объекта goods создадим наследника ( через proto ), который используется для описания плитки (для пола, дорожек, стен...) название - tile (tile.js). Объект tile дополните свойствами 

               material (string)
               type (string)
               size_x (mm)
               size_y (mm)
               outdoor (boolean)
               color (string)
               cover (string)
               images (array) - массив изображений ['pic1', 'pic2']
Дополните объект двумя методами 
                метод imperialSize () который возвращает массив [this.size_x, this.size_y] преобразованных размеров в дюймы ( в одном дюйме 25.4 миллиметров). Значения должны быть округлены до сотых.
                метод isSquare () - возвращает true или false в зависимости от того, является ли плитка квадратная.
Создайте данный объект (протестируйте самостоятельно - создайте, заполните, проверьте методы).

Task 3.На основании объекта goods создадим наследника ( через proto ), который используется для описания ламината (для пола, дорожек, стен...) название - laminate(laminatejs). Объект laminate дополните свойствами 

            color (string)
            thickness (mm)
            suitable (string)
            finish (string) ex: gloss, rustic, smooth, textured...
            flitting_mechanism (string) ex 1click2go, drop loc, fold down...
            guaranteed (int)
            resistance (int) ex: 31, 32, 33, 54
            unit_coverage (m2) ex: 2.005
В данном объекте размеры одинаковы для всех и равны: 
            size_x = 1292 (mm)
            size_y = 194 (mm)
Дополните объект двумя методами метод getBoard(square) int принимает параметр площади в m2 и возвращает количество досок ламината. метод getUnitCount(square) int - принимает параметр в m2 и возвращает количество полных упаковок (округление в большую сторону). Создайте данный объект (протестируйте самостоятельно - создайте, заполните, проверьте методы).

Task 4. Мы упустили важный параметр у каждого товара есть уникальный номер артикул. добавим art в объект goods. Убедитесь, что он стал доступен и в laminate и в tile.
